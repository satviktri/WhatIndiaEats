# What India Eats

A clean, reader-friendly, bilingual (English/Hindi) web page that explains key takeaways from the ICMR–INDIAB-21 diet study in plain language.

## 📄 Paper

This project summarizes and links to the Nature Medicine paper:

- **Dietary patterns in India and metabolic risk (ICMR–INDIAB-21)**  
  DOI: [10.1038/s41591-025-03949-4](https://doi.org/10.1038/s41591-025-03949-4)

## ✨ What this repository contains

- `indiaeats.html` — a single-page bilingual explainer with:
  - language toggle (English / हिंदी)
  - accessible structure and skip-link
  - editorial-style typography and layout
  - citation block linking to the original paper

## 🚀 Quick start

No build step is required.

1. Clone the repository.
2. Open `indiaeats.html` in your browser.

Or run a tiny local server:

```bash
python3 -m http.server 8000
```

Then open: <http://localhost:8000/indiaeats.html>

## 🧭 Purpose

The goal is to make important nutrition and metabolic-health research from India easier to understand for a broad audience, without replacing the original scientific publication.
